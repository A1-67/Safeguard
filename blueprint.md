# **App Name**: Guardian Angel

## Core Features:

- Emergency SMS: Send emergency SMS with location to saved contacts.
- Location Detection: Detect and include current GPS location in SMS.
- Siren Sound: Play a loud, looping siren sound when the emergency button is activated, toggle siren on/off when pressed again.
- Emergency Dialer: Open phone dialer with 911 pre-entered, ready to call.
- Bluetooth Trigger: Detect Bluetooth media button presses to trigger emergency function.
- Contact Management: Settings page to add/remove emergency contacts.
- Testing Tools: Settings page includes siren and location detection test tools.

## Style Guidelines:

- Primary color: Vibrant sky blue (#72B5FF) to evoke a sense of urgent help.
- Background color: Deep midnight blue (#1A1A2E) for a minimalistic and safe aesthetic.
- Accent color: Electric purple (#BE72FF) to add contrast on the settings pages.
- Font: 'Inter' (sans-serif) for all text, providing a clean and modern interface. Note: currently only Google Fonts are supported.
- Center a large circular 'EMERGENCY' button for immediate access.
- Simple, minimalist icons in settings for contact management and testing tools.
- Subtle animation when the emergency button is pressed, indicating activation.
